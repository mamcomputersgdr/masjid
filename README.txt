► Using PWABuilder (Easiest, FREE):
  1. Host your index.html on any website
     (GitHub Pages is free)
  2. Go to: https://www.pwabuilder.com
  3. Enter your website URL
  4. Click "Build My PWA"
  5. Download the Android APK package
  6. Install the APK on your phone!

► Using Capacitor (For developers):
  1. Install Node.js on PC
  2. npm install -g @capacitor/cli
  3. Copy your HTML into a project
  4. npx cap add android
  5. npx cap open android
  6. Build APK in Android Studio
