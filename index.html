☁️ Google Drive Auto-Sync
🔴 Not connected to Google Drive

Connect your Google account to automatically save all receipts and expenses to Google Drive. Works in background — every time you save a receipt it syncs instantly when online.

Get free from console.cloud.google.com → APIs → Credentials → OAuth 2.0

How to get Client ID (free, 5 mins):
1. Go to console.cloud.google.com
2. Create project → Enable Google Drive API
3. Credentials → Create → OAuth 2.0 Client ID
4. App type: Web application
5. Authorised origin: your GitHub Pages URL
6. Copy Client ID → paste above
